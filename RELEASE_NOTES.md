# v0.1.1 发布说明

发布日期：2026-07-17

这是 `design-child-reading-adventures` 的玩法节奏多样性补丁发布。

## 修正内容

- 将“玩法多样”定义为孩子实际进入的主要互动语法／决策循环，而不是不同标题、故事细节、道具、版面或成品。
- 四站以上的手册不允许相邻重复同一种循环，同一循环最多两次，至少四类；八站以上至少六类。
- 家长方案新增节奏核验，儿童手册出图前检查新增“同一种读后做一做换皮”拦截。
- 新增 V02 十站回归，能拒绝重复的“读后做／摆一摆”流程，同时保留文本依赖、开放控制、连续阅读和非评分原则。

## 验证摘要

- 既有固定自审 smoke：5/5 任务、19/19 必需标准通过；自动失败与回归失败均为 0。
- I01、V01、P01、T01、O01 与 V02 定向回归通过。
- 本版本新增 V02 为静态／自审回归，尚未完成隔离的新代理 forward test 或新的真实渲染成品测试。

## 已知限制

- 具体图书设计仍取决于合法、可靠且与版本匹配的材料。
- 孩子明确偏好重复某种媒介时可以保留，但不能把同一决策循环的换皮误称为多样。
- 本 Skill 不保证学习效果，也不替代专业发展或安全建议。
