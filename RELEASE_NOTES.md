# v0.1.0 发布说明

发布日期：2026-07-16

这是 `design-child-reading-adventures` 的首次中文开源发布。

## 主要能力

- 按请求选择 `ideas`、`plan` 或 `full`，避免小需求被扩成整本手册。
- 通过 A-D 证据等级限制情节、人物、引文、章节和版本细节。
- 只有核验匹配版本分页后才写原书页码。
- 完整模式默认分开交付成人方案和儿童绘本式手册。
- 核心活动必须让刚读到的内容改变玩法，并通过移除图书测试。
- 开放活动要求儿童真正控制过程或结果，允许停止、换媒介、不分享和不留痕。
- 内置儿童材料、动作、媒体、公开分享与 AI 使用的安全边界。

## 验证摘要

- 固定自审 smoke：5/5 任务、19/19 必需标准通过。
- 自动失败：0；回归失败：0。
- 五项 2026-07-16 定向回归通过。
- 真实双文件 DOCX/PDF 生成与逐页渲染检查通过，但相关个性化产物不公开。

## 已知限制

- 尚无隔离的新代理 forward test。
- 具体图书设计质量取决于合法、可靠且与版本匹配的材料。
- 本 Skill 不保证学习效果，也不替代专业发展或安全建议。
