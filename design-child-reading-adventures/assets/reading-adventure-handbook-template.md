# 《__________》阅读冒险绘本

> Child-only artifact. Build one coherent, playable picturebook world. The map is the contents and route. Each location is a two-page picturebook spread: Page A is the story scene and invitation; Page B is the playable scene or creation surface. Add Page C only as optional extra space.

## 绘本视觉总则

- 先让孩子看见一个世界、角色或奇怪事件，再看见说明。
- 使用一个与作品气质相连的场景系统和有限色板；可加入一个反复出现、说话很少的向导角色或视觉母题。
- 用大色块、简单形状、表情、气泡、地图、门、印章、收藏物和可移动视觉对象制造趣味。可爱来自角色和事件，不来自堆满装饰。
- 让插图承担至少一种功能：讲故事、表达情绪、呈现选择、承载玩法或指路。删掉没有功能且挤占阅读／游戏空间的装饰。
- 默认把页面的大部分留给场景和操作。孩子文字只保留大标题、短情境、读书入口、2—4个起步动作和小型下一门提示。
- 使用大号、清晰、对比足够的文字；图标必须有文字／形状辅助，不能只靠颜色区分。文本尽量保持可编辑，不把关键说明烘焙进图片。
- 明确划分标题、场景、说明、游戏、印章和导航区域；任何角色或装饰不得压住文字、页码或下一门。

## 只属于孩子的内容边界

孩子绘本中不出现：证据等级、版本核验过程、阅读诊断、家长操作说明、比例公式、合规核验、成人评语或安全论证。必要安全提醒改写为简短、可执行的孩子语言；完整理由放在家长方案。

## 封面｜这本书里藏着一个世界

- 大标题和一句任务邀请。
- 一个主视觉角色／物件／未完成事件。
- 一句“翻开就能做”的动作，不要求先读规则。
- 可选：孩子名字栏，但不要求真实姓名或公开身份。

## Start Now 跨页｜先让故事动一下

在封面后立即放一个三分钟内能开始、无需准备的真实故事门。用一个醒目的场景和一件可以摆、选、猜、发声、移动或想象的事情开场。允许只读、跳过或停下。

## 冒险地图跨页｜同时也是目录

把每站画成故事世界中的地点、星球、门、岛、房间或角色。每站重复同一编号、名称、图标／形状、颜色和绘本内部页码，并留一个可选印章／涂色点。原书页码只在对应版本分页已核验时出现；否则完全省略，不留空格。地图不是顺序表：写明可以跳、停、回头或推开任意门。不要另做成人式目录表。

## Location Page A｜故事场景：门开了

### 1. 场景标题

使用短而有动作感的标题，如“脚印没有说话”“十个太阳不肯下班”。标题应来自已核验情节，不复制长段原文。

### 2. 图像主场景

用角色、物件、天气、地点或冲突占据主要视觉区域。可让向导角色用一个短气泡指出异常，但不要解释完故事。

### 3. 发生了什么

用几句孩子能读懂的文字交代一个悬念、冲突、变化或未完成事件。少剧透，只保留启动阅读和游戏所需的信息。

### 4. 读这里

按证据只选择一种入口：

- 对应版本分页已核验：可写准确原书页码／范围；
- 内容已核验、分页未核验：只写可靠的章节／小节／情节门，不出现页码或页码空格；
- 没有准确内容：不写具体情节，改为“选择今天想读的一段，读到自己想停”。

保护连续阅读。可放一个“藏着读的小秘密”，但必须可忽略。不要让孩子填写、猜测或跨版本寻找页码。

### 5. 现在让它动起来

先用一句孩子能懂的话说清“从刚读的内容带什么进来”，再让它真的改变路线、规则、选择或结果。给一个可见的游戏对象和2—4个可任选、可换序、可忽略的起步动作：摆、搭、选、换规则、走路线、发声、扮演、试验或想象。让孩子看得见至少三个真正会改变玩法的控制钮，其中至少一个是“目标／规则／结局”；颜色、名字和装饰不能凑数。附一个很小的例子并标明“不是标准答案”。不要用角色名字、故事标题或装饰图案冒充阅读连接，也不要把最后一句“你可以修改”接在一整套必做配方后面。

## Location Page B｜可玩的画面：这一页归你

让这一页本身成为游戏空间，而不是空白作业纸。根据玩法只选一个主结构：

- 可改造的地图／迷宫／路线；
- 可摆放物件的场景、舞台、房间或地形；
- 可补全、换位、折叠、排序或改规则的图像；
- 声音／动作路径；
- 大幅创作场景加极少量可选文字；
- 作品收藏框、印章点或本地照片／录音名称栏。

页面只放：一句可选起笔／起玩提示、一个停止条件、两个或三个具名下一门，以及“任意故事门／今天只读／停下”。明显写出可以换材料、换做法、改规则或结局、拒绝扩展，也可以什么痕迹都不留；不要把这些选择藏在家长说明里。下一门可以使用绘本内部页码；只有对应版本分页已核验时才附原书页码。不要重复整套规则、评分、日志或反思问卷。

## Optional Page C｜世界还想长大

只有扩展绘画、搭建记录、地图、拼贴或长故事确实需要空间时才添加。Page A-B 已经必须完成选择、阅读、玩、可选痕迹、求助桥和下一门；Page C 不能藏必做步骤或导航。

## 夜空／收藏页｜留下什么都算

把可选痕迹设计成星星、印章、收藏格、物件影子或小展览。孩子可以写日期、画一个符号、贴作品名称，也可以保持空白。禁止排名、连续打卡、数量比较和成人签字。

## 任意故事门

提供同样的两页绘本跨页：Page A 用“改变一个物件／能力／决定／世界规则”生成游戏；Page B 是宽松场景。孩子可以换书、换门、只读或停止。

## 出图前检查

- 家长内容是否全部留在另一份方案中？
- 第一眼是否看到场景／角色／可玩的对象，而不是说明卡片？
- 每张图是否在讲故事、承载玩法或指路？
- 孩子能否在90秒内做出第一个动作？
- 每个核心游戏是否都说清了“刚读到的什么”会改变玩法？移走这本书或换一本无关的书后，游戏若仍基本不变，是否已经重做？
- 被计为开放的任务是否列得出至少三个会改变过程／结果的儿童控制钮，并包含可改变或放弃的目标、规则或结局？若只剩颜色、命名或最后一步的象征性选择，是否已经重做？
- 文字是否足够大、对比清楚，并且不只靠颜色导航？
- 角色、气泡、印章、页码和导航是否完全不重叠？
- 所有原书页码是否都有对应版本的分页证据？没有证据时是否连空白页码栏也已删除？
- A-B 是否自包含完整循环，Page C 是否仍可完全跳过？
