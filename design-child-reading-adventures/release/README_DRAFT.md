# 儿童阅读冒险设计 Skill

`design-child-reading-adventures` 是一个面向家长、教师与儿童阅读设计者的 Codex Skill。它先核验图书与版本证据，再把可确认的内容转化为儿童主导、与阅读真实相连的活动点子、成人阅读方案或完整的阅读冒险手册。

## 它解决什么问题

- 不凭模型记忆编造页码、章节、人物、引文或版本细节。
- 不把普通作业换个可爱标题，而是让刚读到的细节真正改变玩法。
- 长线手册不靠换标题、道具或成品假装多样；孩子实际进入的推理、取舍、试验、角色回应或规则改造等决策循环需要有节奏地变化。
- 在完整模式下，将家长方案与孩子手册分开交付。
- 孩子可以选择路线、媒介、规则、停止点以及是否留下作品。
- 对松散材料、动作、拍照、公开分享和 AI 使用提供安全与隐私边界。

## 三种输出模式

- `ideas`：少量、边界清晰的阅读活动点子。
- `plan`：家长／教师使用的阅读路线、节奏、活动与调整方案。
- `full`：成人方案与儿童绘本式阅读冒险手册两个独立交付物。

## 快速开始

把仓库中的 `design-child-reading-adventures/` 复制到：

```text
~/.codex/skills/design-child-reading-adventures/
```

刷新 Codex 后，可以这样使用：

```text
请使用 design-child-reading-adventures，为一本儿童读物设计两周阅读冒险。
孩子9岁，每次能读30分钟，喜欢推理和手工。请先确认版本证据；如果没有准确分页，不要写原书页码。我要分开的家长方案和孩子手册。
```

## 重要边界

- 只有在核验同一版本分页后才写原书页码。
- 不复制大段受版权保护文本，也不使用未经授权的图书副本。
- 活动不依赖成人评分、签字、持续提问或公开发布儿童信息。
- 本 Skill 提供阅读体验设计支持，不保证学习效果，也不替代特殊教育、发展或安全方面的专业意见。

## 反馈

如发现编造图书细节、错误页码、与阅读无关的游戏、隐藏作业、儿童自主性不足或版面碰撞，请提交 Issue，并附上去标识化的请求、证据等级、预期行为与实际行为。不要上传儿童姓名、学校、照片、声音、私人阅读记录或受版权保护的整本图书。
