# 展示计划

## 目标

用不含真实儿童信息和受版权图书正文的合成案例，展示证据核验、输出模式选择、阅读依赖玩法、开放控制与成人／儿童分离。

## 示例输入

```text
为一本用户提供了版权页和相关节选的儿童冒险小说设计两周阅读冒险。
孩子9岁，每次30分钟，喜欢推理和搭建。需要可打印的家长方案与孩子手册。
```

## 示例输出应展示

1. 明确证据等级和不能确认的内容。
2. 不写未经核验的原书页码。
3. 每个核心游戏都让刚读到的细节改变路线、规则、选择或后果。
4. 孩子至少控制三个会改变过程或结果的维度。
5. 家长方案与孩子绘本式手册分开。
6. 提供无材料、安全、私密和“今天只读”的路线。

## 不公开

- 真实儿童姓名、学校、照片、声音或家庭记录。
- 用户生成的个性化手册原件。
- 受版权保护的书页、译文或整本文件。
- 本地路径、密钥、私有工作流和内部评审原始材料。
